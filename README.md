# HỆ THỐNG QUẢN LÍ THƯ VIỆN

## Giới thiệu

Dự án **HỆ THỐNG QUẢN LÍ THƯ VIỆN** được xây dựng nhằm hỗ trợ công tác quản lí thư viện trong môi trường đại học. Hệ thống giúp số hóa các nghiệp vụ cơ bản như quản lí sách, bạn đọc và mượn – trả sách,....

## Mục tiêu

* Quản lí danh mục sách một cách khoa học
* Hỗ trợ theo dõi bạn đọc
* Thực hiện nghiệp vụ mượn – trả sách chính xác
* Giảm sai sót, tiết kiệm thời gian cho cán bộ thư viện

## Chức năng chính

* Quản lí sách (thêm, sửa, xóa, tìm kiếm)
* Quản lí bạn đọc
* Quản lí mượn – trả sách
* Thống kê, báo cáo cơ bản

## Công nghệ sử dụng

* Ngôn ngữ: Python
* Môi trường làm việc: Visual Studio Code
* Quản lí mã nguồn: Git & GitHub

Dự án được thực hiện với mục đích học tập, rèn luyện kỹ năng phân tích, thiết kế và lập trình hệ thống quản lí cho sinh viên.
